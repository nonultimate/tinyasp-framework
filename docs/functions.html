include(String filename [, once])

Dynamic include file with repeating inclusion support, like include_once function in PHP.
Note: use it with eval() (JavaScript will not execute it as global variables).

Parameter
filename The file to include.
once [Optional]Whether the file should be included only once, default is true.
Return String
Example // Include a file once
eval(include("include.asp"));
// Include a file any times
eval(include("any.asp", false));
eval(include("any.asp", false));

redirect(var url)

Redirect URL. If url is a number, it will redirect to error page with HTTP state code (0, 403, 404).

Parameter
url The URL of the location to redirect, or HTTP state code.
Return void
Example
// Redirect a URL
redirect("/index.html");
// Goto 403 error page
redirect(403);
// Goto 404 error page
redirect(404);
// Goto error page
redirect(0);

require(String lib)

Parameter

Return void
Example // Import file lib/mail/mail.asp

// Import file lib/text/utf16.asp

// Require app/lib/test.asp
require("Test");
