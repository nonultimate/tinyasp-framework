$.Cookie

    Class for manipulating cookies.


Methods

get(String name)

Get a cookie.

Parameter
name The cookie name to get.
Return String

set(String name, var value [, String expire [, String path [, String domain [, Boolean secure ]]]])

Set a cookie.

Parameter
name The name of the cookie.
value The value of the cookie.
expire [Optional]Seconds expires from now.
path [Optional]The path in the server the cookie will be available.
domain [Optional]The domain that the cookie will be available.
secure [Optional]The cookies should only be transmitted over HTTPS.
Return void

remove(String name)

Remove a cookie.

Parameter
name The cookie name to remove.
Return void

removeAll()

Remove all cookies.

Return void
