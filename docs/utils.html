Pager(Integer total, Integer size, String query)

    Pager class.

Parameter
total The total number of records.
size The size of records to display per page.
query The query string for the current page or records offset.
Example
Controller code
var total = model.count();
var pagesize = 10;
var pager = new Pager(total, pagesize, "page");
$.view.assign("pager", pager);
 View code
<asp:pager var="pager" />
