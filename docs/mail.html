$.Mail

    Class for sending mails. Require JMail component.


Properties

The instance of JMail object.

The mail server.

Parameter
filename The file to add.
inline [Optional]The file is inline or not.
Return Integer
    Returns the content ID, useful only for mails in HTML.

mail(String from, String to, String subject, String body [, Boolean isHTML])

Send a single mail. Use alias send() instead.

Parameter
from The mail sender.
to The one you send to.
subject The subject of the mail.
body The content of the mail.
isHTML Plain text or HTML.
Return Boolean
Example $.Mail.send("test<test@test.com>", "hello<hello@hello.com>", "Mail test", "Send mail sucessfully!");

send(String subject, String body [, Boolean isHTML])

Send the mail.

Parameter
subject The subject of the mail.
body The content of the mail.
isHTML Plain text or HTML.
Return Boolean
Example // Set the mail server
$.Mail.setServer("pop.test.com", "user", "password");
// Add a recipient
$.Mail.addRe("test@test.com", "test");
// Add a carbon copy recipient
$.Mail.addCC("hello@test.com");
// Send the mail in HTML
$.Mail.send("Hello", "<h1>Hello from Tinyasp!</h1>", true);
